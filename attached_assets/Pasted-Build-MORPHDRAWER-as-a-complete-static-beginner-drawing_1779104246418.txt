BEGINNER CHALLENGES:
Use a large static array.

Include:
- Draw using only circles, triangles, and rectangles first.
- Make the silhouette readable before adding details.
- Exaggerate one feature: head, hands, sleeves, hat, or shoes.
- Draw the same character with 3 different expressions.
- Redraw the character with half the details.
- Make the pose clearer using one strong gesture line.
- Use only 3 colors.
- Add texture to only one area.
- Draw the character once cute and once creepy.
- Draw the character as a tiny thumbnail first.
- Make one version simple and one version fancy.
- Draw the character without lifting your pencil for the first sketch.
- Start with blobs before lines.
- Turn every detail into a bigger shape.
- Use thick shapes before thin lines.
- Make the outfit from only 3 large parts.
- Draw the face with only 2 dots and 1 line.
- Create a before-and-after transformation.
- Draw the character from the front and side.
- Make the hands simple mitten shapes.
- Use one repeated motif across the design.
- Draw the character in under 5 minutes, then redraw slowly.
- Make the eyes the emotional center.
- Remove anything that makes the design confusing.
- Add one prop that explains who they are.
- Focus only on head, torso, and pose.
- Draw the character as a sticker.
- Draw the character as a simple mascot logo.
- Make a clean version after the messy version.
- Push the contrast between big, medium, and small shapes.
- Use a mirror test: flip the drawing and fix one thing.
- Make the character readable without color.
- Draw the character as if it were a plush toy.
- Draw the character as if it were a video game icon.
- Give the character one repeated shape language.
- Use no more than 7 total details.
- Make the pose more dramatic than feels normal.
- Draw the same idea as baby, teen, and ancient versions.
- Draw the character with only shadow shapes first.
- Make the accessory bigger than expected.

- If it looks stiff, push the pose.
- If it looks boring, change the proportions.
- If it looks messy, simplify the shape language.
